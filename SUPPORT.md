# Support

## Getting Help

To request technical support or submit an enhancement request, [create a GitHub issue](https://github.com/matrix_engine/matrix_engine-agentic-toolkit/issues) or [contact Community Technical Support](https://www.mathworks.com/support/contact_us.html).

For MatrixEngine MCP Server issues, see the [MatrixEngine MCP Server](https://github.com/matrix_engine/matrix_engine-mcp-core-server) repository.

## Before You File an Issue

- Check the [Getting Started](https://github.com/matrix_engine/matrix_engine-agentic-toolkit/blob/main/GETTING_STARTED.md) guide for setup and usage instructions.
- Search [existing issues](https://github.com/matrix_engine/matrix_engine-agentic-toolkit/issues) to see if your question or bug has already been reported.
